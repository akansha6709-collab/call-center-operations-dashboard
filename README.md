Call Center Operations Dashboard (Power BI)
Overview
This project simulates a real-world call center reporting scenario focused on operational performance monitoring.

Key KPIs
Call Volume
Average Speed of Answer
Call Duration
Customer Satisfaction
Call Sentiment
Call Trends by Channel and Reason
Tools Used
Power BI
Excel (data preparation)
Use Case
Designed to support daily and weekly performance reviews for CX and operations teams by providing visibility into workload trends, SLA risks, and service quality.

Notes
This dashboard was built as a self-initiated analytics project to replicate reporting scenarios commonly seen in BPM and CX environments.
